# 動物診断ガイド（100種類対応版）

このドキュメントでは、100種類の動物それぞれの特徴と、どの選択肢を選べばその動物が診断結果として表示されるかを説明します。

## 診断の仕組み

1. **11個の質問**に回答します
2. 各選択肢には**特性（traits）**が紐づいています
3. 全ての回答から特性を集計し、各動物の特性とマッチングします
4. **最もスコアが高い動物**が診断結果として表示されます

## スコアリングの仕組み

各動物には4つの特性が割り当てられています。ユーザーが選んだ回答から集めた特性と、各動物の特性をマッチングし、一致する特性の数がスコアになります。

例：
- ユーザーが選んだ特性: `social`, `active`, `cheerful`, `social`, `active`, `cheerful`, ...
- イヌの特性: `loyal`, `social`, `active`, `friendly`
- スコア: `social`と`active`が複数回マッチするため、高いスコアになります

最もスコアが高い動物が診断結果として表示されます。
